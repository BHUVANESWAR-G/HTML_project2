## 📌 Events Website

### 📖 About

A simple HTML-based **Events Webpage** showcasing upcoming events, gallery, and contact details.
Includes banners, images, and external links for a clean event presentation. 

---

### ✨ Features

* 📅 Upcoming events section
* 🖼️ Image gallery display
* 🔗 External fashion store link
* 📞 Contact information section
* 🎨 Basic styling using HTML

---

### 🛠️ Tech Stack

* HTML5
* Basic inline CSS
